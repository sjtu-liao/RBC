# Rayleigh-Bénard convection
A sustained turbulence, i.e. the two-dimensional (2D) turbulent Rayleigh-Bénard (RB) convection, is solved by means of "Clean Numerical Simulation" (CNS) and the traditional direct numerical simulation (RKwD), respectively, and the related movie can be downloaded here.

For more details, please see:

S. Qin, S. Liao, Large-scale influence of numerical noises as artificial stochastic disturbances on a sustained turbulence. *Journal of Fluid Mechanics* (2022), vol. 948, A7. (doi: 10.1017/jfm.2022.710)
